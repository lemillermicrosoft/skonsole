Semantic Kernel is an open-source SDK that allows developers to easily integrate AI services like OpenAI, Azure OpenAI, and Hugging Face with conventional programming languages such as C# and Python. It enables the creation of AI applications that combine the best of both worlds. Semantic Kernel is lightweight and offers features for configuring, running, and customizing the kernel, as well as using variables, planner, and memory. It can be used to create AI-first experiences with prompts, skills, memories, and planners.
